# NZA Simulate

Building energy simulation and analytics platform powered by EnergyPlus.

React + FastAPI + SQLite + EnergyPlus

## Setup

1. Clone to `C:\Dev\nza-simulate` (local, NOT OneDrive)
2. Install EnergyPlus (v25.2.0+)
3. `cd frontend && npm install --force`
4. `pip install -r requirements.txt`
5. Double-click `go.bat` to launch
